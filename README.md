# Sitio Web - Escuela de Suboficiales de la Policía de Corrientes

Proyecto de desarrollo web institucional para la Escuela de Suboficiales de la Policía de Corrientes.

## Tecnologías utilizadas

- HTML
- CSS
- Bootstrap
- JavaScript

## Características

- Diseño responsive
- Componentes reutilizables (navbar y footer)
- Organización modular del proyecto
- Galería de autoridades
- Sección de formación policial

## Estructura del proyecto

/css → estilos  
/js → scripts  
/components → navbar y footer reutilizables  
/imagenes → recursos gráficos
